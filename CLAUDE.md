# Project: Pavocard Website

## Overview
This is a fintech website for Pavocard, a platform that allows users in Nigeria and Ghana to sell gift cards and receive cash.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Design
- Based on Figma design
- Modern fintech style (like Stripe/Flutterwave)
- Clean UI, responsive layout

## Coding Rules
- Use reusable components
- Use semantic HTML for SEO
- Keep code clean and readable
- Avoid unnecessary complexity

## Animations
- Use Framer Motion
- Smooth, subtle, premium animations
- Avoid excessive motion

## SEO
- Use proper metadata
- Use heading hierarchy (H1–H3)
- Optimize for performance

## Notes
- Do not break existing code
- Ask before making major changes
